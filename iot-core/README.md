# iot-core

Plain AWS IoT Core variant of the watchman — the holly-002 comparison
experiment (same topics and payloads as the Greengrass version in
`../greengrass/watchman/`, but the script owns its own MQTT connection,
certificates, service management, and updates).

Empty until that experiment starts. See the PoC report for the
Greengrass-vs-IoT-Core decision context.
